Act like Minions는 같은 외형을 지닌 미니언들 사이에서 플레이어를 잡아내는 게임입니다.  

# How to Run
1. 프로젝트 다운로드 혹은 클론
2. 'ActlikeMinions.exe' 실행
3. 같은 LAN에 연결된 플레이어와 플레이

# Skills
- Host and Client : Unity
- Network Library : Mirror
- Modelling & Art : Blender
